Why did we make this?

We built this project for CIS*4250 in Winter 2021. It includes a web app hosted at http://cis4250-06.socs.uoguelph.ca/ and an Electron desktop app with the same functionality. This is a useful tool for current and future U of G students, staff, faculty, program counsellors and more!

The goal of the project is to be like Webadvisor - but way better. Our interface gives users an easy way to view, search and visualize course offerings at the University of Guelph. Users can search through courses using various useful fields, then view a table and graph of their results. The graphs help visualize the prerequisite dependencies between courses using a tree layout.

How did we make this?

We used a few different tools to bring this project together. We built a web scraper using Python to scrape Webadvisor for updated course offerings each day. We also built a parser (also using Python) to parse and store additiona; information about each course. Our web server was built using NGINX and our desktop app runs on Electron. Our search API is built with Node.JS and Express, while our front end is simple HTML/CSS and JavaScript. To see our project documentation and code, click the link below!
